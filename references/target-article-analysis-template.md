# Target Article Analysis Template

Use this when the user provides target articles, target journal examples, or asks
for a journal-specific writing model.

## Article Set

For each article:

- citation or file name
- journal and year
- paper type
- target section analyzed
- why it is comparable

## Section Model Table

| Unit | Text span | Function | Language markers | Grammar choices | Notes for user's paper |
|---|---|---|---|---|---|
| 1 | paragraph/sentence | e.g. establishes importance | phrase family | tense, passive, modal | adapt/avoid |

## Synthesis

After analyzing articles, synthesize:

- common opening functions
- common closing functions
- optional functions
- average paragraph count or length where useful
- figure/table placement patterns
- accepted level of background
- how contribution and limitation are expressed
- title/abstract conventions

## Transfer to User Draft

Create:

1. Local section model.
2. Paragraph plan.
3. Phrase/function bank.
4. Language-risk notes.
5. Draft or revision.

## Guardrails

- Do not copy distinctive sentences from target articles.
- Use target articles to model function and convention, not to reproduce prose.
- If target articles conflict, prefer the closer journal, newer article, and
  closer paper type.
- Clearly mark any inference from the target set.
